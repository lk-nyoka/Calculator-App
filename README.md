# Python Calculator 

A sleek, functional calculator application built with Python

## Features
* **User-friendly GUI:** Clean interface for easy calculations.
* **Standalone Application:** Runs as an '.exe' file.
* **Standard Operations:** Handles basic arithmetic with error handling.

## How to Download & Run
You do not need Python installed to use this calculator. 

1. Go to the [Releases Section](https://github.com/lk-nyoka/Calculator-App/releases). 
2. Click on the latest release (v1.0).
3. Under **Assets**, click 'Calculator2.0.exe' to download.
4. Double-click the file to run the calculator.

## Built With
* **Language:** Python 3.12
* **Compiler:** PyInstaller
